# todo-list-app

A full-stack to-do list web app built with React/Next.js, TypeScript, Tailwind CSS, and deployed on Vercel.

Users can add, delete, and mark tasks as complete.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as complete

## Tech Stack

- React/Next.js + TypeScript
- TailwindCSS
- Vercel (deployment)
